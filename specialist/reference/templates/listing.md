# Template — Listing Description

Output structure for the listing job. Three audience variants — pick one based on the audience target from `checklists/listing-intake.md` (or produce stacked variants if multiple audiences are required).

## When to consult this file

Per `rules.md`: every listing job. The specialist follows the relevant variant below.

## Market-specific disclosures

Before producing the listing, check `region/market.md` for a **"Buyer-disclosure conventions"** section. If present, surface the relevant items in the listing proactively — they reflect what the local buyer pool expects to see. Examples vary by market: tsunami/elevation context in coastal Thailand; lease registration in leasehold markets; HOA/Mello-Roos in U.S. markets; Tama 38 status in Israeli urban markets. If the section is absent, skip — the market has no codified conventions yet.

## Language rule

Default: produce the listing in **one** language at a time — the audience's working language.

- **Local-buyer audience (Variant A):** local language by default.
- **English-speaking buyer (Variant B):** English by default.
- **International investor / lifestyle (Variant C):** the buyer's working language by default; ask the agent if unclear (English is the safe default for most non-Anglo international buyers).

**Stacked output is on request, not default.** When `region/market.md` lists 2 working languages and the audience straddles both pools (e.g., Hebrew + English in Jerusalem for a property that will market to both locals and Anglo buyers), produce stacked listings if the agent asks for it. Per `rules.md`: side-by-side or stacked, never interleaved within a sentence.

When `region/market.md` lists 3+ working languages (common in resort and gateway markets), do NOT produce all variants by default — that's wasteful. Pick one based on the named audience; offer to produce additional variants on request.

## Buyer-type → variant routing

| Buyer type | Variant to use |
|---|---|
| Local end-user | A |
| Returning diaspora / relocating expat (Type 4 in `frameworks/international-buyers.md`) | B |
| English-speaking non-local buyer (no investment angle) | B |
| Yield investor (Type 1) | C, full investment-grade framing |
| Capital-appreciation investor (Type 2) | C, full investment-grade framing |
| **Lifestyle / legacy international buyer (Type 3)** | **B scaffold + foreign-buyer considerations block + agent-notes block from C.** Don't lead with an Investment Summary table for Type 3 — they are not investing primarily, they are buying a holiday home or legacy asset. The yield offset (if relevant) is one bullet, not the headline. |

For ambiguous cases (e.g., a Type 3 buyer who *also* cares about rental yield offset), use the Type 3 hybrid: B scaffold + selective C blocks. State which blocks you're including and why.

## Variant A — Local buyer (lifestyle anchor)

Use when the audience is a local end-user. Voice: warm, lifestyle-led.

### Required sections

**Headline**
Short, evocative, in local language. 5–10 words.

**Lifestyle hook**
1–2 sentences. What kind of life this property offers.

**Room-by-room walk**
3–5 sentences in flowing prose. Layout, light, materials, the texture of the space.

**Neighborhood paragraph**
2–3 sentences. Character, amenities within walking distance, transit, daily life.

**Features (bullets)**
3–5 distinguishing features. Not boilerplate.

**Asking price**

**Viewing call-to-action**

---

## Variant B — English-speaking buyer (relocation / non-local)

Use when the audience is an English-speaking buyer who is not local — typically a returning diaspora, expat, or relocator. Voice: warm, lifestyle-led, with extra context an outsider needs.

### Same scaffold as Variant A, plus:

- **Transit and access details** — how to get around; distance to airport
- **Schools** — public / private / religious options nearby, with brief evaluation
- **Services for English-speakers** — community institutions, healthcare in English, key services
- **Walkability assessment** — what's accessible without a car
- **Currency** — local denomination + USD/EUR if useful for the audience

---

## Variant C — International investor (investment-grade)

Use when the audience is an international investor (yield, capital appreciation, or hybrid). Voice: professional, data-led. **Always consult `frameworks/international-buyers.md`** to pick the right buyer-type framing within this variant.

### Required sections (in this order)

**Headline**
Property type + asset class + location anchor. Investment-grade framing, not lifestyle.

**Investment summary (bullets)**
- Asking: price + currency
- Current rent / income (if applicable)
- Gross yield (if applicable)
- Hold thesis: yield / capital appreciation / hybrid

**Property facts (table)**
| Field | Value |
|---|---|
| Type | SFH / multi-unit / commercial |
| Size | m² or sqft |
| Layout | units, BR/BA |
| Year built / renovated | year / scope |
| Ownership status | clean Tabu / fee simple / etc. (per `region/regulations.md`) |
| Occupancy | occupied / vacant / mixed |

**Thesis (bullets)**
3–4 bullets. Why this property, why now, why this market. Anchored in the buyer-type framing from `frameworks/international-buyers.md`.

**Foreign-buyer considerations**
Surface only — verify with local counsel and accountant. Pull from `region/regulations.md`:
- Tax exposure (purchase tax bracket, capital gains, foreign surcharges)
- Documentation (power-of-attorney, AML / source-of-funds, identity)
- Restrictions, if any

**Exit liquidity**
1–2 sentences on comparable transaction history and hold-to-sale norms in this segment.

**Next steps for due diligence**
3–5 numbered items the buyer should verify. Specific to this property.

**Closing caveat**
"Not legal or tax advice. Buyer should verify with local counsel and accountant."

---

## Always include — Notes for the agent (Variant C only)

After the listing, append a section addressed to the **agent** — not buyer-facing. This includes:

1. What to clarify with the buyer's intermediary before the showing
2. Source-of-funds path expectations (banks, jurisdictions)
3. Hold thesis to confirm (yield vs. flip — pitch differs)
4. Decision pace and decision-makers — set internal expectations

This section is for the agent's prep, not for forwarding to the buyer.
