# Template — Buyer–Property Fit Summary

Output structure for the matching job. Always follow this structure.

## When to consult this file

Per `rules.md`: every matching job.

## Required sections (in this order)

### 1. Buyer profile summary

One paragraph. Compact summary of the buyer that matters for fit decisions. Include: household, budget (with stretch), key constraint or anchor, end-user vs. investor.

### 2. Listings ranked (table)

| Rank | Listing | Overall fit | Key reason |
|---|---|---|---|
| 1 | [identifier] | Clear / Acceptable / Friction / Blocker | [1 sentence — what makes this top or bottom] |
| 2 | [identifier] | [score] | [reason] |
| 3 | [identifier] | [score] | [reason] |

Order by overall fit, best to worst. Score scale per `frameworks/buyer-fit.md`.

### 3. Detailed scoring — top pick (table)

| Dimension | Score | Notes |
|---|---|---|
| Budget | [score] | [1 sentence] |
| Location | [score] | [1 sentence] |
| Type/size | [score] | [1 sentence] |
| Lifestyle | [score] | [1 sentence] |
| Friction | [score] | [1 sentence] |

Five dimensions per `frameworks/buyer-fit.md`.

### 4. Top recommendation

1–3 sentences. Which listing to show first, and the buying-decision insight (what closes the deal — schools, the view, the workspace, the community walk, etc.). Make the agent's first showing strategic, not just rank-1.

### 5. Honest tradeoff

1–2 sentences. What the buyer compromises on if they go with the top pick. Visible upfront — not surfaced at the showing.

### 6. Don't dismiss yet *(conditional)*

Include only if the #2 listing has a real case the agent shouldn't write off. 1–2 sentences. Otherwise omit.

### 7. No-good-fit case *(conditional)*

Include only if NO listing in the candidate set is a good fit. Recommend the agent go back to sourcing rather than show. Do not produce a top recommendation.

## Honesty principle (from frameworks/buyer-fit.md)

Always call out friction. Never inflate a fit to push a property. The agent reads this output before deciding which property to show first — over-scoring costs them a wasted showing and a bruised relationship.
