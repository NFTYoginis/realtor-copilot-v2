# Showing-note template

The output structure for a showing note. Concise, structured, citable. The agent should be able to scan it in 30 seconds and copy/paste sections into a CRM or follow-up email.

## Structure

```
# Showing note — [Property address] — [Date]

**Buyer:** [identifier]
**Attendees:** [who was at the showing]
**Agent:** [name, if multi-agent team]

## Reaction summary
[2–3 sentences. Overall read on this buyer for this property.]

## Positive signals
*(from transcript)*
- [Specific feature, room, view — what they said or did]
- [Repeat for each]

## Friction
*(from transcript)*
- [Specific concern + the agent's interpretation if added]
- [Practical objection: commute, schools, HOA, taxes, parking, condition]

## Comparisons made
*(from transcript)*
- [Other property they referenced + the comparison they drew]

## Questions raised
*(from transcript)*
- [Question + what the question reveals about their concern]

## Logistics
- **Timeline cues:** [...]
- **Decision-makers not present:** [...]
- **Money cues:** [financing/contingencies/sell-first]

## Agent's read
*(agent input, not from transcript)*
- **Seriousness for this property:** [1–5]
- **Next step:** [specific action]
- **Homework before next conversation:**
  - [ ] [Specific item 1]
  - [ ] [Specific item 2]

## Tags for later retrieval
- **Buyer:** [identifier]
- **Property:** [identifier]
- **Friction themes:** [normalized tags — e.g., `noise`, `ground-floor-bedrooms`, `hoa-cost`, `commute`, `kid-bedrooms-too-small`]
```

## Rules for producing this note

- **Cite source per line.** "from transcript" vs. "agent's read." Don't blend. The agent must know which is which on re-read.
- **Don't invent.** If the transcript doesn't mention schools, the schools field stays empty or marked "not raised."
- **Friction is the most valuable section.** Buyers tell you "no" before they tell you "yes." Capture the no clearly, with the underlying concern when stated.
- **Tags must be normalized.** Use the same tag for the same concept across notes — e.g., always `hoa-cost` (not `hoa-fees`, `monthly-dues`, `condo-cost`). When in doubt, propose a tag and ask the agent to confirm.
- **Length.** As long as the showing requires, no longer. A 5-minute walk-and-out tour produces a 5-line note. A 90-minute tour with a serious buyer produces a fuller note.
