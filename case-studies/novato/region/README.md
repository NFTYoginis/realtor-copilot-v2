# Case Study — Novato Region Pack (POPULATED)

This folder contains a populated Novato / Marin County, CA region pack. To use the specialist for Novato:

1. Copy the contents of this folder over `realtor-copilot/specialist/reference/region/`.
2. Drop the `specialist/` folder into your Claude Project.

Files in this pack:
- `market.md` — Novato buyer segments, seasonality, dynamics, pricing magnitudes
- `regulations.md` — California / Marin County regulations, Prop 13, FIRPTA, Mello-Roos
- `neighborhoods.md` — Major Novato neighborhoods (Pleasant Valley, Hamilton, Bel Marin Keys, Marin Country Club, Old Town)
- `contracts.md` — California residential lease and CAR sale agreement norms
- `glossary.md` — California-specific technical terms (English-monolingual market, light glossary)
