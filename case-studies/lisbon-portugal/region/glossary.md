# Glossary — Lisbon, Portugal (Portuguese ↔ English)

Common Portuguese real-estate and legal terms an agent encounters and a buyer might ask about. Use this file when a term appears in a query whose meaning could be ambiguous to an English-default speaker.

## Property types

| Portuguese | English / context |
|---|---|
| **Apartamento** | Apartment / flat |
| **T0 / T1 / T2 / T3 / T4** | Studio / 1BR / 2BR / 3BR / 4BR. The number is bedrooms (quartos), not rooms total. T2 = 1 living room + 2 bedrooms. |
| **Moradia** | House / townhouse (free-standing or semi-detached) |
| **Moradia em banda / em banda contínua** | Row house / terraced house |
| **Quinta** | Estate / farm with land — often rural, varies in scale |
| **Loja** | Commercial / shop space |
| **Escritório** | Office space |
| **Terreno** | Land parcel (building or unbuilt) |
| **Duplex** | Two-floor apartment |
| **Triplex** | Three-floor apartment |
| **Penthouse / cobertura** | Top-floor apartment, often with terrace |

## Property features

| Portuguese | English / context |
|---|---|
| **Quarto** | Bedroom |
| **Sala** | Living room |
| **Cozinha** | Kitchen |
| **Casa de banho / WC** | Bathroom (full / half) |
| **Suite** | En-suite bedroom (bedroom with attached bathroom) |
| **Terraço** | Terrace |
| **Varanda** | Balcony |
| **Marquise** | Glassed-in balcony or veranda; common in Lisbon stock — sometimes counted as additional living area, sometimes not. Verify per listing. |
| **Arrumos** | Storage room |
| **Garagem** | Garage / parking |
| **Lugar de estacionamento** | Parking spot |
| **Elevador** | Elevator. Critical to call out — older central Lisbon buildings often don't have one, materially affecting price and buyer pool. |
| **Rés-do-chão (R/C)** | Ground floor |
| **Andar** | Floor / story |
| **Vista rio / Vista Tejo** | River view (Tagus) |
| **Vista castelo** | Castle view (São Jorge) |
| **Logradouro** | Private back yard / outdoor space at ground level |
| **Pátio** | Patio / courtyard |
| **Mansarda / águas-furtadas** | Attic / mansard (often the converted top floor) |

## Title and registration

| Portuguese | English / context |
|---|---|
| **Caderneta Predial** | The tax-registry document from Finanças. Identifies the property and gives its tax-assessed value. |
| **Certidão de Teor** *(or **Certidão Permanente**)* | The title-registry document from Conservatória do Registo Predial. Confirms ownership, encumbrances, mortgages. |
| **Conservatória do Registo Predial** | The land/property registry office. |
| **Finanças** | The tax authority (informally — the local Finanças office). |
| **Notário / Cartório Notarial** | Notary / notary's office. Mediates the escritura. |
| **Escritura** | The deed-signing event; the closing in U.S. terms. |
| **Licença de Utilização** | Permitted-use license issued by the municipality (residential vs. commercial vs. mixed). |
| **Ficha Técnica de Habitação** | Construction record (post-2004 buildings). |
| **Certificado Energético** | Energy-performance certificate, required at sale. |

## Contract terms

| Portuguese | English / context |
|---|---|
| **Reserva** | Reservation agreement, optional pre-contract step. |
| **CPCV — Contrato de Promessa de Compra e Venda** | Promissory contract; the binding "we will sell and buy" agreement. The U.S. analogue is a purchase agreement after contingency removal. |
| **Sinal** | Earnest money / down payment paid at CPCV. Typically 10–30% of price. |
| **Condomínio** | Condominium / building HOA. Also refers to the building's association of owners. |
| **Assembleia de Condóminos** | The building's annual or extraordinary owners' meeting; minutes carry decisions. |
| **Quotas / despesas de condomínio** | Building HOA dues. |

## Taxes and fees

| Portuguese | English / context |
|---|---|
| **IMT — Imposto Municipal sobre Transmissões** | Property transfer tax, paid by buyer. |
| **IS — Imposto do Selo** | Stamp duty. |
| **IMI — Imposto Municipal sobre Imóveis** | Annual property tax. |
| **AIMI — Adicional ao IMI** | Additional property tax for high-value holdings. |
| **Mais-valias** | Capital gains (on disposal). |
| **NIF — Número de Identificação Fiscal** | Taxpayer number. Required to transact. |
| **NHR — Non-Habitual Resident** | Tax-residency regime; rules and applicability have changed over time. |
| **Representante fiscal** | Tax representative — non-resident buyers/owners often appoint a Portuguese accountant. |

## Visa / immigration

| Term | English / context |
|---|---|
| **Golden Visa** | Investment-visa program. **Closed for residential-property purchases in October 2023.** |
| **D7** | Passive-income / retirement visa. |
| **D8** | Digital-nomad visa. |
| **HQA / Tech / R&D pathways** | High-skill / research visa tracks. |

## Short-term rental

| Portuguese | English / context |
|---|---|
| **Alojamento Local (AL)** | Short-term rental regulatory regime. License required. Issuance frozen in many central Lisbon parishes (containment zones). |
| **Containment zone** *(zona de contenção)* | Parish where new AL issuance is frozen, making existing licenses valuable. |

|---|---|

| **Mediador imobiliário** | Real-estate agent. |
| **AMI** | The licensing body for real-estate agencies in Portugal — `licença AMI` is the agency's license number. |
| **Casa para arrendar** | Rental property (long-term). |
| **Casa para venda** | Property for sale. |
| **Avaliação** | Valuation / appraisal. |
| **Vistoria** | Inspection / walk-through. |
| **Habitação própria permanente** | Primary residence — distinct from second home for tax purposes. |
| **Recuperar / requalificar** | Renovate / rehabilitate (especially for historic stock). |
| **Prédio** | Building (the apartment building or property as a whole). |
| **Fração** | Apartment unit within a building, in legal language. |

## How the specialist uses this glossary

When the user uses a Portuguese term not in the working English vocabulary, the specialist looks it up here first to ensure it's being used as the user intended — *before* responding. Particular care for terms with English false friends: `marquise` is not a covered porch / awning in this market; `pátio` is private outdoor space, not a public square.
