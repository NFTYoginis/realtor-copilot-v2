# Case Study — Jerusalem Region Pack (POPULATED)

This folder contains a populated Jerusalem region pack. To use the specialist for Jerusalem:

1. Copy the contents of this folder over `realtor-copilot/specialist/reference/region/`.
2. Drop the `specialist/` folder into your Claude Project.

Files in this pack:
- `market.md` — Jerusalem buyer segments, seasonality, dynamics, foreign-buyer activity, pricing magnitudes
- `regulations.md` — Mas Rechisha, Mas Shevach, Arnona, Tabu vs. Khevra Meshakenet, foreign-buyer rules
- `neighborhoods.md` — Major Jerusalem neighborhoods (Old Katamon, Baka, German Colony, Talbiya, Rehavia, Old City Jewish Quarter, Pisgat Ze'ev)
- `contracts.md` — Israeli residential lease and sale agreement norms
- `glossary.md` — Hebrew real estate terminology
